# 2012 Avengers - Who are you?
You will be presented a list of questions which will try to guess which original Avenger you are. Please answer in "yes" or "no" for it to work. 

Here is the hero list: 
Iron Man
Captain America
Thor
Black Widow
Hawkeye
Hulk
